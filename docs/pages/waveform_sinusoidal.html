1. Sine wave¶

$$ \large x(t)=A sin(2\pi f t + \phi)=A sin(\omega t + \phi) $$

where,

  • $A$ = The aplitude;
  • $f$ = The ordinary frequency;
  • $\omega$ = The angular frequency;
  • $\phi$ = The phase.
In [3]:
A = 1
f = 220
omega = 2*np.pi*f
phi = 0

x = A*np.sin(omega*t + phi)
